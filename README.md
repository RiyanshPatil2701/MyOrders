# My Orders - Android Assignment
A clean and native Android application implementing the Orders management tracking screen UI with interactive system components.
## Features
**Dynamic Lists (RecyclerView):** Engineered a decoupled vertical list pattern to safely render customized layout rows representing distinct consumer transactions.
**Dynamic Interaction:** Custom runtime shifting highlights for bottom navigation text/icons and category selection segments.
**Responsive Architecture:** Fully built using ConstraintLayout with responsive guidelines to maintain layout pixel symmetry across different screen dimensions.
**Component Modularity:** Decoupled status filters mapped through dynamic state views.
# Tech Stack
**Language:** Java
**UI Structure:** XML (ConstraintLayout, LinearLayout, RecyclerView)
**Target SDK:** API 34+
